# VinFast Project - Docs Index

Current architecture is Next.js fullstack.

- App UI: Next.js pages
- API: Next.js API routes (`frontend/pages/api`)
- Database: PostgreSQL (Docker service)
- Public port: `3000`

Useful files:

- `README.md`: quick overview
- `START_HERE.md`: fast start
- `docker-compose.yml`: run stack
- `Dockerfile`: app image build
- `.env.example`: environment variables
- `run.sh`: local run helper
- `install.sh`: dependency install helper
